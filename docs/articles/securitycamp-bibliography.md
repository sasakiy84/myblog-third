# セキュリティキャンプ資料一覧

セキュリティキャンプ関連の記事を集めた索引記事です。
どちらかといえば、司書的な興味から資料の列挙を試してみたくてやりました。思ったより検索に引っかからないのが多いなと感じました。講義名などを検索キーワードに入れると新しい記事がヒットすることがありました。あとは、列挙しようとすると結構時間を喰いますね。

来年度以降応募課題に解答するときや、自習用の資料を探すとき、セキュリティ関係のブログを探すときなどに活用してもらうことを想定しています。

- 時系列はそあまり気にしてません
- 可能な限り網羅的な列挙を心掛けましたが、抜け漏れ等あるはずです
    - 特定キャンプの参加記事を書いているブログは、他のキャンプに参加していないか確認しています
    - リンクが切れてる記事は、Internet Archive で閲覧できる記事もありましたが、一応除いています
- そのうち、それ単体でまとまっている記事や、応募課題晒しの記事、特定分野の資料などのメタデータを整えていきたいです
